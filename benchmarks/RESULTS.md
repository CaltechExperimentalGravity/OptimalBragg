All benchmarks on Apple Silicon, `coatingDev` conda env (Python 3.14, NumPy 2.3, Numba 0.63).
Run `python benchmarks/bench_oldvsnew.py` from the repo root to reproduce.

All benchmarks use identical config: **14 pairs (29 layers), popsize=40**.
Generated by `bench_oldvsnew.py` with raw data in `results_master.json` / `results_refactor.json`.

## Full Pipeline Profiling (14 pairs, popsize=200, workers=1)

Run `python benchmarks/profile_full_pipeline.py` from the repo root to reproduce.

### Optimization (production config)

| Metric       | Before (workers=-1) | After (workers=1 + precompute) | Change    |

| Throughput   | ~10k evals/sec      | 18,255 evals/sec              | **~1.8x** |
| Per-eval     | ~100 us (IPC+eval)  | 54.8 us                       | **~1.8x** |
| Final cost   | 1.912               | 1.912                         | identical |
| Func evals   | ~559k               | 558,870                       | —         |

### MCMC (5000 samples)

| Phase            | Time       |
|------------------|------------|
| Sampler setup    | 0.66 sec   |
| Array precompute | 0.5 ms     |
| Eval loop        | 0.11 sec   |
| Per-sample       | 22 us      |

### What Changed (this round)

- `workers=1`: eliminated 62-110 us IPC overhead per eval (was dwarfing 22 us eval cost)
- `precompute_misc()`: pre-builds n array, active set, wavelength arrays once before devo
- Inlined `brownianCost` in hot path (avoids function call + `np.sum` dispatch overhead)
- Vectorized MCMC array setup (eliminated 5000 `np.copy` calls)
